# JavaScriptレベル7

プログラミングの世界には、効率よく、かつ整理されたコードを書くための<span class="info">オブジェクト指向</span>という強力な考え方があります。

その中心となるのが、このレベル7で学ぶ<span class="info">「クラス（Class）」</span>です。

これまでの学習では、変数や関数を組み合わせて一つひとつ処理を書いてきました。

しかし、プログラムが大きくなるにつれて、<span class="info">「似たようなデータをまとめて扱いたい」</span><span class="info">「一度作った仕組みを別の場所でも再利用したい」</span>という場面が増えてきます。

クラスは、いわば<span class="info">「データと処理をセットにした設計図」</span>です。

このカリキュラムでは、クラスの作り方から、それを元に実体（インスタンス）を生み出す方法、さらには既存のクラスをパワーアップさせる<span class="info">「継承」</span>や、データを守る<span class="info">「カプセル化」</span>といった一歩進んだテクニックまでを、ステップバイステップで習得していきます。

## 内容
JavaScriptレベル7では以下のテーマを取り扱います。

* クラス基礎
* コンストラクタ
* メンバ変数
* メソッド
* 継承
* オーバーライド
* 静的メンバ

「ECMAScript2015」以降に準拠していますが、本カリキュラムでは上記の分野のみを取り扱います。

## 対象
JavaScriptレベル7は以下のような方を対象にしています。

* JavaScriptの学習が初めての方
